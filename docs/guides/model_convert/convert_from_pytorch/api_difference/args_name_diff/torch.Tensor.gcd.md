## [ 仅参数名不一致 ]torch.Tensor.gcd
### [torch.Tensor.gcd](https://docs.pytorch.org/docs/stable/generated/torch.Tensor.gcd.html#torch.Tensor.gcd)
```python
torch.Tensor.gcd(other)
```

### [paddle.Tensor.gcd](https://www.paddlepaddle.org.cn/documentation/docs/zh/develop/api/paddle/Tensor__upper_cn.html#gcd-x-y-name-none)
```python
paddle.Tensor.gcd(y, name=None)
```

两者功能一致，仅参数名不一致，具体如下：
### 参数映射

| PyTorch                          | PaddlePaddle                 | 备注                                                   |
|----------------------------------|------------------------------| ------------------------------------------------------ |
|  other  |  y  | 输入的 Tensor ，仅参数名不一致。                                     |
