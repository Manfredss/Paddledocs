## [ 仅参数名不一致 ]torch.Tensor.lcm
### [torch.Tensor.lcm](https://docs.pytorch.org/docs/stable/generated/torch.Tensor.lcm.html#torch.Tensor.lcm)
```python
torch.Tensor.lcm(other)
```

### [paddle.Tensor.lcm](https://www.paddlepaddle.org.cn/documentation/docs/zh/develop/api/paddle/Tensor__upper_cn.html#lcm-x-y-name-none)
```python
paddle.Tensor.lcm(y)
```

两者功能一致且参数用法一致，仅参数名不一致，具体如下：

### 参数映射

| PyTorch       | PaddlePaddle | 备注                                                   |
| ------------- | ------------ | ------------------------------------------------------ |
| other  |   y   | 表示输入的 Tensor ，仅参数名不一致。 |
